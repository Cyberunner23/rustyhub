Fields

Internal http client.

Base URL to the API, can be modified to use the enterprise API.

user_agent: String

Value used for the User-Agent key in request headers.

authentication: Auth

Optional authorization token, acquired from https://github.com/settings/tokens, should be set to None when purely accessing endpoints requiring no authentication.
